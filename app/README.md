# RememberCall 리멤버콜 — 자녀용 앱 프로토타입

> 자녀가 부모님의 인지 건강을 관리하는 모바일 앱 UI 프로토타입.  
> HTML/CSS/JS 단일 파일로 구현된 인터랙티브 목업입니다.

## 화면 구성 (12개 스크린)

앱은 `<section class="screen">` 단위로 화면을 전환하며,  
`go('screenId')` 함수로 이동합니다. 상태는 `localStorage`에 저장됩니다.

### 진입 흐름

```
스플래시(splash)
  └─ 온보딩1(ob1) → 온보딩2(ob2) → 온보딩3(ob3)
       └─ 목소리 등록(voice) → 가족 추억(history) → 통화 스케줄(schedule)
            └─ 홈(home) ← [이미 온보딩 완료 시 직접 진입]
```

### 메인 탭 (하단 탭바)

| 탭 | 화면 ID | 내용 |
|----|---------|------|
| 홈 | `home` | 주간 케어 현황, 오늘 통화 상태, 인지 지표, 위험 알림, 하이라이트, 케어 팁 |
| 리포트 | `reportlist` | 월간 달력 + 날짜별 리포트 목록 |
| 추세 | `trend` | 주간/월간 오답률 바차트, 전월 대비 지표 |
| 내 정보 | `me` | 프로필, 구독 현황, 설정 항목 |

### 상세 화면

| 화면 ID | 진입 경로 | 내용 |
|---------|-----------|------|
| `report` | 리포트 목록 → 날짜 탭 | 답변 지연·망설임·정답률·카테고리 분석 |
| `share` | 리포트 상세 → 공유 버튼 | 링크 공유·PDF 내보내기·가족 초대 |
| `alert` | 홈 알림 카드 / 벨 아이콘 | 위험 신호 상세 + 병원 찾기 버튼 |
| `voice` | 온보딩 / 내 정보 | 목소리 재녹음 화면 |
| `history` | 온보딩 / 내 정보 | 가족 추억 목록 + 추가·수정·삭제 |
| `schedule` | 온보딩 / 내 정보 | 통화 시간·요일 설정 |

## 주요 기능 상세

### 가족 추억 관리
- 추억 항목: 카테고리(출생지·직장·자녀 태명·기념일 등) + 내용 텍스트
- CRUD: 추가(바텀 시트) / 수정(편집 아이콘) / 삭제
- 저장: `localStorage` (`rc_memories` 키)
- 기본 데이터 4건 내장 (최초 실행 시 자동 로드)

### 통화 리포트
- 7개 날짜 샘플 데이터 내장 (`_dailyReports` 객체)
- 날짜별 지표: 답변 지연(초), 망설임 빈도, 정답률, 기억 망각 카테고리
- 달력에서 날짜 탭 → 해당 날짜 리포트 렌더링
- 6월 5일: 주의(warn) 상태로 강조 표시

### 홈 동적 콘텐츠
- **대화 하이라이트**: 6개 샘플 중 랜덤 노출
- **케어 팁**: 90개 팁 중 랜덤 노출 (운동·식품·대화법·활동 카테고리)

### 온보딩 상태 관리
- 완료: `localStorage.setItem('rc_onboarded', 'true')`
- 앱 진입 시 완료 여부 확인 → 홈 또는 스플래시로 분기
- 내 정보 > "온보딩 다시 보기" 로 초기화 가능

## 디자인 시스템

웹사이트(`website/`)와 동일한 베이지 팔레트 사용.

### 컬러

| 변수 | 값 | 용도 |
|------|----|------|
| `--cream` | `#F6EFE2` | 기본 배경 |
| `--cream-2` | `#F0E7D6` | 카드 내부, 보조 배경 |
| `--caramel-deep` | `#9E7C4D` | CTA 버튼, 강조 |
| `--brown` | `#4A3D2E` | 기본 텍스트 |
| `--brown-2` | `#6B5C49` | 보조 텍스트 |
| `--brown-3` | `#9E8B72` | 레이블, 설명 |
| `--alert` | `#B5563C` | 위험 알림 |
| `--alert-bg` | `#F3E0DA` | 위험 알림 배경 |

### 레이아웃
- 디바이스 컨테이너: `width: 390px` (iPhone 14 기준)
- 배경: `#3A2E25` (어두운 브라운) — 데스크톱에서 폰 목업처럼 보이게
- 탭바: `position: absolute; bottom: 0` 고정

### 주요 컴포넌트

| 클래스 | 설명 |
|--------|------|
| `.card` | 흰 배경 둥근 카드 (기본) |
| `.card-dark` | `--caramel-deep` 배경 강조 카드 |
| `.alert-card` | 위험 알림용 분홍 카드 |
| `.litem` | 리스트 아이템 (탭 가능) |
| `.setrow` | 설정 행 (토글 포함) |
| `.seg` | 세그먼트 컨트롤 (주간/월간 등) |
| `.sheet-backdrop` | 바텀 시트 오버레이 |
| `.toggle` | on/off 토글 스위치 |
| `.days` | 요일 선택 토글 버튼 그룹 |
